# 2D Escape Room Game (Java)

A 2D Java game where the player controls a character, avoids falling boxes from a dropper, and escapes the room to progress to the next level.

## Features
- Player movement and controls
- Falling obstacle mechanics
- Collision detection
- Level progression
- Game GUI and input handling

## Tech Stack
- Java
- Object-Oriented Programming
- Ant build system

## Key Classes
- Player
- Dropper
- GameEngine
- InputHandler
- GameGUI
- Tile

## What I Learned
- Building game logic using Java
- Applying object-oriented programming
- Debugging gameplay issues
- Structuring classes for interactive applications
